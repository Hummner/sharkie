Class: PoisonBottle

PoisonBottle(minX, maxX)

Represents a poison bottle collectible in the game. Inherits from MovableObject.

Constructor

new PoisonBottle(minX, maxX)

Creates a new PoisonBottle instance at a random x position within a range.
Parameters:
Name Type Description
minX number The minimum x coordinate where the bottle can appear.
maxX number The range added to minX to define the maximum x coordinate.
