Class: Ammo

Ammo(x, y, otherDirection)

Class representing ammo (e.g., a bubble trap projectile). Extends MovableObject to inherit movement and drawing capabilities.

Constructor

new Ammo(x, y, otherDirection)

Creates an instance of Ammo.
Parameters:
Name Type Description
x number The initial X position of the ammo (usually the character's x).
y number The initial Y position of the ammo (usually the character's y).
otherDirection boolean Direction flag; true if ammo moves in the opposite direction.
